# 101QA Archives

This is a static clickable website for the Queer Public History Microfestival project.

## How to open locally
1. Unzip the folder.
2. Open `index.html` in Chrome, Safari, or Firefox.
3. Click the episode cards to open source popups.

## How to make it live
Fastest options:
- Netlify: drag the whole folder into Netlify Drop.
- Vercel: create a new project and upload/import the folder.
- GitHub Pages: push the folder to a repository and enable Pages.

## What to edit before final submission
- Confirm exact archive collection names, dates, box numbers, and folder numbers for each source.
- Revise the museum caption so it sounds like your own final analysis.
- Keep the AI disclosure or modify it to match what you actually used.
- Replace placeholder source notes with final archival citations.
